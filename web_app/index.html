This is a network of all NBA players who played during the 2024-25 regular season. The size of each player node is a combination of games played/started and win shares. The width of an edge from a player to team is a measure of impact through games played/started, win shares, and average box plus-minus. The width of an edge between teammates is a measure of impact using minutes played together, win percentage, and effective net rating.
